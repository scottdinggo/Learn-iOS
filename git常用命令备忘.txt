git 配置：

git config --global user.name “scottdinggo”
git config --global user.email "scottdinggo@gmail.com"

git 常用命令：

git help <command>  # 显示command的help
git show            # 显示某次提交的内容
git show $id

git co -- <file>    # 抛弃工作区修改
git co .

git add <file>      # 将工作文件修改提交到本地暂存区
git add .           # 将所有修改过的工作文件提交本地暂存区

git rm <file>       # 从版本库中删除文件

git reset <file>    # 从暂存区恢复到工作区
git reset --head    # 恢复最近一次提交的状态
